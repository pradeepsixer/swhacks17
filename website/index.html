SW HACKS 2017 Project aimed at providing safe aviation . Project uses AWS, SQS and Open Sky API

PROBLEM STATEMENT

.  

The collision of aircraft with flying objects in space are incidents of frequent occurrences in recent times that pose serious risks to air travel safety. We aimed at solving this issue by using the Open Sky API to collect flight coordinates in real time.
